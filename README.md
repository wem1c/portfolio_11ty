## Getting Started

### Prerequisites

- [pnpm](https://pnpm.io/) (or equivalent)

### Installation

1. Clone the repo

   ```sh
   git clone https://codeberg.org/wem1c/portfolio_11ty.git
   ```

   or

   ```sh
   git clone https://github.com/wem1c/portfolio_11ty.git
   ```

2. Install dependencies

   ```sh
   pnpm install
   ```

3. Run the dev script

   ```sh
   pnpm run dev
   ```
